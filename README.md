# Sabão Sustentável – Site de Assinatura

Projeto de um site simples e responsivo para divulgação e venda de um produto sustentável,
com foco em assinatura mensal e impacto ambiental positivo.

## Objetivo
Facilitar a venda do produto por meio de uma landing page acessível via navegador,
direcionando o cliente para contato via WhatsApp ou pagamento via Pix.

## Tecnologias Utilizadas
- HTML5
- CSS3 (Mobile First)
- JavaScript
- GitHub Pages

## Funcionalidades
- Site responsivo (desktop e mobile)
- Botão de contato direto com WhatsApp
- Apresentação do produto e proposta sustentável
- Divulgação do modelo de assinatura

## Aplicação
Projeto aplicado em um negócio real de produção artesanal de sabão ecológico,
com reaproveitamento de óleo e gordura usados, contribuindo para a redução de impactos ambientais.
